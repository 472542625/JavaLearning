exists(not exists)
select * from user where exists (select 1);等同于select * from user;
exists返回true的时候前面执行，exists返回false 的时候前面不执行

in(not in)
select * from user where userId in (1, 2, 3);等效于
select * from user where userId = 1 or userId = 2 or userId = 3;
